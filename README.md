# Thesis Project – Integration of Machine Learning Models to Predict Demand in Inventory Optimization
This repository contains the code developed for the master's thesis:

Integration of Machine Learning Models to Predict Demand in Inventory Optimization

The project investigates whether machine-learning-based demand forecasting combined with uncertainty quantification and stochastic inventory optimization can improve platelet inventory management compared to traditional moving-average approaches.

## Project Overview
Platelets are highly perishable blood products with a shelf life of only five days. Accurate demand forecasting and inventory planning are therefore critical to reducing shortages, minimizing waste, and maintaining high service levels.

This project implements a complete decision-support framework consisting of:
* Demand forecasting using statistical and machine learning models
* Uncertainty quantification through prediction intervals and bootstrap simulation
* Scenario generation using clustering techniques
* Inventory optimization using a mixed-integer programming model

The case study is based on historical platelet demand data from the Finnish Red Cross Blood Service.
